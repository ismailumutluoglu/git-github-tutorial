## 📋 **En Sık Kullanılan Komutlar**

| Komut                   | Anlamı                              |
| ----------------------- | ----------------------------------- |
| `git init`              | Yeni git deposu başlatır            |
| `git clone <repo-url>`  | Var olan bir projeyi klonlar        |
| `git status`            | Değişiklik durumunu gösterir        |
| `git add <dosya>`       | Dosyayı commit'e hazırlar           |
| `git commit -m "mesaj"` | Değişiklikleri kaydeder             |
| `git log`               | Commit geçmişini gösterir           |
| `git diff`              | Değişiklikleri listeler             |
| `git branch`            | Branch'leri listeler                |
| `git checkout <branch>` | Branch değiştirir                   |
| `git merge <branch>`    | Branch birleştirir                  |
| `git pull`              | Uzak repodan değişiklik çeker       |
| `git push`              | Değişiklikleri uzak repoya gönderir |

---

## 🎯 **Pratik Başlangıç**

```bash
git init
git add .
git commit -m "İlk commit"
git remote add origin https://github.com/kullaniciadi/depoadi.git
git push -u origin main
```

> 🔎 Komutları terminalde denemekten çekinme!
